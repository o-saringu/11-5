プログラム復習コンテンツ

配列-応用（マウスを追跡する）

サンプルコードの結果

サンプルコード

　　 int num=60;

　　 int[] x=new int[num];

　　 int[] y=new int[num];

　　 　

　　 void setup(){

　　 　size(240, 120);

　　 　noStroke();

　　 }

　　　 　

　　 void draw(){

　　 　background(0);

　　 　for(int i=x.length-1; i>0; i--){

　　 　　x[i]=x[i-1];

　　 　　y[i]=y[i-1];

　　 　}

　　 　x[0]=mouseX; 　

　　 　y[0]=mouseY;

　　 　for(int i=0; i　　 　　fill(i*4);

　　 　　ellipse(x[i], y[i], 40, 40);

　　 　}

　　 }

説明文

　配列-応用では、マウスを追跡するプログラムを配列を使用して作成している。

　2つの配列があり、マウスのx座標の配列とy座標の配列が記録される。

　配列には、常に60フレーム分のマウスのxy座標が残っていて、古い座標は新しい座標に置き換えられていく。

　新しい座標を記録する前に、配列内の各要素を1つずつ前から後ろへずらすことで常に最初の要素を新座標の記録場所として確保している。

　フレームごとに60個の座標を描いて、配列の動きを視覚化する。
